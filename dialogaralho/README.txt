##########################3
			Como Instalar
###########################
1. Estar logado como ROOT
2. Este script substituirá 
instalações anteriores
3. Execute
	# bash install.sh

	thales.faggiano@sp.senai.br

		  Como remover
#############################
1. Estar logado como ROOT
2. Este script removerá 
instalações anteriores
3. Execute
	# bash remove.sh
4. Qualquer erro contate:
	thales.faggiano@sp.senai.br
