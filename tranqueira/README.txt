#######################################
             Como instalar
#######################################
1. É necessário estar logado como ROOT
2. Execute o comando:
	# bash install.sh
3. O script fará o backup antes de fazer 
a instalação nos diretórios necessários.
4. Em caso de falhas contate:
	thales.faggiano@sp.senai.br
#######################################
             Como Remover
#######################################
1. É necessário estar logado como ROOT
2. Execute o comando:
	# bash remove.sh
3. O script removerá tudo associado ao
	programa tranqueira
4. Em caso de falhas contate:
	thales.faggiano@sp.senai.br
